# RPL-315251020007-AmosReynaldi

Repository ini dibuat untuk memenuhi tugas Praktikum Rekayasa Perangkat Lunak (RPL)Pertemuan 12 & 13

## Deskripsi

Repository ini berisi:

1. Rencana pemeliharaan perangkat lunak (_maintenance plan_) yang mencakup kategori
   Corrective, Adaptive, Perfective, dan Preventive.
2. Dokumentasi arsitektur perangkat lunak untuk studi kasus **CineTix**
   (aplikasi pemesanan tiket bioskop online), meliputi deskripsi sistem, quality
   attributes, gaya arsitektur, logical view, dan physical view.

## Daftar Isi

- `maintenance/`
  - `maintenance-plan.md` — Rencana pemeliharaan perangkat lunak
- `architecture/`
  - `deskripsi-sistem.md` — Deskripsi sistem CineTix
  - `quality-attributes.md` — Atribut kualitas yang menjadi prioritas sistem
  - `arsitektur.md` — Pola/gaya arsitektur yang digunakan beserta alasannya
  - `logical-view.md` — Gambaran komponen/modul logis sistem
  - `physical-view.md` — Gambaran infrastruktur fisik/deployment sistem
